How to use the integrated browser color picker

Open developer tools panel

In case you are not aware of color pickers I will bring them to your attention. The color picker is a tool used very often in web development and not just by designers. In order to use it you have to open developer tools panel. To open it usually you have to access main menu and there you will find developer tools in almost all browsers. Or you can press F12. Most browsers should open developer tools.

Select a HTML element contaning text

If this is your first visit there, don't panic. Ignore most of the stuff you see there. Usually you will see a panel called "Console" which should be mostly white. This is not what we need. Look for a panel called "Elements". After opening "Elements" you should see some HTML in the main view. Just click on any of these tags and on the right side you should see a panel with CSS rules. For the moment just ignore them. First one should usually be named element.style {}

Change the color of the text live

One cool thing you can do with developer tools is to change the page without actually saving or refreshing. This is done on the fly and the changes will be lost on refresh Try typing there color hit "enter" and than type "red", enter again. The way you enter CSS instructions in dev tools is a bit cumbersone unfortunately, it takes a bit of practice. Now you should see some red text there. Feel free to try the other notations aswell. Notice that when you type wrong syntax a striketrough will appear, indicating that the rule is not active. We will talk more about dev tolls in the upcoming chapters. Right now remember F12 to open dev tools and the CSS panel for live edits and experiments.
